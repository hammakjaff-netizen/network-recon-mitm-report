# Network Reconnaissance and Man-in-the-Middle Attack Simulation Report

This repository contains a professional technical assessment report based on a controlled virtual lab environment.

## Summary
The report covers:
- Network discovery with Nmap
- ARP spoofing and man-in-the-middle concepts
- Wireshark packet capture analysis
- SSL/TLS downgrade troubleshooting
- Vulnerability scanning and findings
